加热 ​

仅加热 ​

苯的磺化 ​

Ph+H2SO4⇌(70∼80°C)ΔPh−SO3H+H2O\rm Ph + H_2SO_4 \xrightleftharpoons[(70\sim80\degree C)]{\Delta} Ph{-}SO_3H + H_2O Ph+H2​SO4​Δ(70∼80°C)​Ph−SO3​H+H2​O

醛基的检验 ​

银氨溶液：

R−CHO+2[Ag(NH3)2]OH→ΔR−COONH4+2Ag↓+3NH3+H2O\rm R{-}CHO + 2[Ag(NH_3)_2]OH \xrightarrow{\Delta} R{-}COONH_4 + 2Ag\downarrow + 3NH_3 + H_2O R−CHO+2[Ag(NH3​)2​]OHΔ​R−COONH4​+2Ag↓+3NH3​+H2​O

新制 Cu(OH)2\rm Cu(OH)_2Cu(OH)2​ （斐林试剂） （由● 蓝色 变为）● 砖红色 沉淀

R−CHO+2Cu(OH)2+NaOH→ΔR−COONa+Cu2O↓+3H2O\rm R{-}CHO + 2Cu(OH)_2 + NaOH \xrightarrow{\Delta} R{-}COONa + Cu_2O \downarrow + 3H_2O R−CHO+2Cu(OH)2​+NaOHΔ​R−COONa+Cu2​O↓+3H2​O

酯的彻底水解（含皂化） ​

以乙酸乙酯为例

CH3COOCH2CH3+NaOH→ΔCH3COONa+CH3CH2OH\rm CH_3COOCH_2CH_3 + NaOH \xrightarrow{\Delta} CH_3COONa + CH_3CH_2OH CH3​COOCH2​CH3​+NaOHΔ​CH3​COONa+CH3​CH2​OH

以苯酚甲酯为例：

Ph−OOCH+2NaOH→ΔPh−ONa+CHOONa+H2O\rm Ph{-}OOCH + 2NaOH \xrightarrow{\Delta} Ph{-}ONa + CHOONa + H_2O Ph−OOCH+2NaOHΔ​Ph−ONa+CHOONa+H2​O

皂化反应书写过于复杂，此处不做赘述

酰胺水解 ​

以乙酰胺为例，酸性条件下：

CH3CONH2+HCl+H2O→ΔCH3COOH+NH4Cl\rm CH_3CONH_2 + HCl + H_2O \xrightarrow{\Delta} CH_3COOH + NH_4Cl CH3​CONH2​+HCl+H2​OΔ​CH3​COOH+NH4​Cl

碱性条件下：

CH3CONH2+NaOH→ΔCH3COONa+NH3↑\rm CH_3CONH_2 + NaOH \xrightarrow{\Delta} CH_3COONa + NH_3\uparrow CH3​CONH2​+NaOHΔ​CH3​COONa+NH3​↑

D-A反应 ​

即1,4加成

CH2=CHCH=CH2+CH2=CH2→ΔC6H10环己烯\rm CH_2{=}CHCH{=}CH_2 + CH_2{=}CH_2 \xrightarrow{\Delta} \mathop{C_6H_{10}}\limits_{\text{环己烯}} CH2​=CHCH=CH2​+CH2​=CH2​Δ​环己烯C6​H10​​

加热+稀硫酸 ​

仅有酯的不完全水解，以乙酸乙酯为例

CH3COOCH2CH3+H2O⇌Δ稀硫酸CH3COOH+CH3CH2OH\rm CH_3COOCH_2CH_3 + H_2O \xrightleftharpoons[\Delta]{\text{稀硫酸}} CH_3COOH + CH_3CH_2OH CH3​COOCH2​CH3​+H2​O稀硫酸Δ​CH3​COOH+CH3​CH2​OH

油脂同理，此处不做展开

加热+浓硫酸 ​

苯的硝化 ​

Ph+HNO3→55∼60°C浓硫酸Ph−NO2+H2O\rm Ph + HNO_3 \xrightarrow[55\sim60\degree C]{\text{浓硫酸}} Ph{-}NO_2 + H_2O Ph+HNO3​浓硫酸55∼60°C​Ph−NO2​+H2​O

制备TNT（甲苯硝化） ​

Ph−CH3+3HNO3→Δ浓硫酸Ph−CH3(−NO2)32,4,6-三硝基甲苯+3H2O\rm Ph{-}CH_3 + 3HNO_3 \xrightarrow[\Delta]{\text{浓硫酸}} \mathop{Ph{-}CH_3({-}NO_2)_3}\limits_{\text{2,4,6{-}三硝基甲苯}} + 3H_2O Ph−CH3​+3HNO3​浓硫酸Δ​2,4,6-三硝基甲苯Ph−CH3​(−NO2​)3​​+3H2​O

醇的卤代 ​

此处使用乙醇+溴化氢演示

CH3CH2OH+HBr→Δ浓硫酸CH3CH2Br+H2O\rm CH_3CH_2OH + HBr \xrightarrow[\Delta]{\text{浓硫酸}} CH_3CH_2Br + H_2O CH3​CH2​OH+HBr浓硫酸Δ​CH3​CH2​Br+H2​O

醇的消去 ​

CH3CH2OH→170°C浓硫酸H2C=CH2↑+H2O\rm CH_3CH_2OH \xrightarrow[170\degree C]{\text{浓硫酸}} H_2C{=}CH_2\uparrow + H_2O CH3​CH2​OH浓硫酸170°C​H2​C=CH2​↑+H2​O

当反应物为乙醇时，必须标注 170°C\rm 170\degree C170°C，但其余醇类不需要

醇的成醚 ​

2CH3CH2OH→140°C浓硫酸C2H5OC2H5+H2O\rm 2CH_3CH_2OH \xrightarrow[140\degree C]{\text{浓硫酸}} C_2H_5OC_2H_5 + H_2O 2CH3​CH2​OH浓硫酸140°C​C2​H5​OC2​H5​+H2​O

当反应物为乙醇时，必须标注 140°C\rm 140\degree C140°C，但其余醇类不需要

酯化反应 ​

此处使用乙醇+乙酸演示

CH3CH2OH+CH3COOH⇌Δ浓硫酸CH3COOCH2CH3+H2O\rm CH_3CH_2OH + CH_3COOH \xrightleftharpoons[\Delta]{\text{浓硫酸}} CH_3COOCH_2CH_3 + H_2O CH3​CH2​OH+CH3​COOH浓硫酸Δ​CH3​COOCH2​CH3​+H2​O

加热+特殊溶剂 ​

水—卤代烃水解 ​

CH3CH2Br+NaOH→ΔH2OCH3CH2OH+NaBr\rm CH_3CH_2Br + NaOH \xrightarrow[\Delta]{H_2O} CH_3CH_2OH + NaBr CH3​CH2​Br+NaOHH2​OΔ​CH3​CH2​OH+NaBr

醇—卤代烃消去 ​

CH3CH2Br+NaOH→ΔC2H5OHH2C=CH2↑+NaBr+H2O\rm CH_3CH_2Br + NaOH \xrightarrow[\Delta]{C_2H_5OH} H_2C{=}CH_2\uparrow +NaBr+H_2O CH3​CH2​Br+NaOHC2​H5​OHΔ​H2​C=CH2​↑+NaBr+H2​O

加热+催化剂 ​

烯炔含氢加成 ​

烯炔含氢加成包括：与 HX\rm HXHX、H2\rm H_2H2​、HCN\rm HCNHCN 加成

炔烃还包括与 H2O\rm H_2OH2​O 加成生成醛，但烯烃不行

CH2=CH2+H2→Δ催化剂CH3CH3\rm CH_2{=}CH_2 + H_2 \xrightarrow[\Delta]{\text{催化剂}} CH_3CH_3 CH2​=CH2​+H2​催化剂Δ​CH3​CH3​

CH2=CH2+HBr→Δ催化剂CH3CH2Br\rm CH_2{=}CH_2 + HBr \xrightarrow[\Delta]{\text{催化剂}} CH_3CH_2Br CH2​=CH2​+HBr催化剂Δ​CH3​CH2​Br

CH≡CH+2H2→Δ催化剂CH3CH3\rm CH{\equiv}CH + 2H_2 \xrightarrow[\Delta]{\text{催化剂}} CH_3CH_3 CH≡CH+2H2​催化剂Δ​CH3​CH3​

CH≡CH+HCN→Δ催化剂CH2=CHCN\rm CH{\equiv}CH + HCN \xrightarrow[\Delta]{\text{催化剂}} CH_2{=}CHCN CH≡CH+HCN催化剂Δ​CH2​=CHCN

CH≡CH+H2O→Δ催化剂CH3CHO\rm CH{\equiv}CH + H_2O \xrightarrow[\Delta]{\text{催化剂}} CH_3CHO CH≡CH+H2​O催化剂Δ​CH3​CHO

生成环己烯

Ph+3H2→ΔPt/Ni(CH2)6环己烷\rm Ph + 3H_2 \xrightarrow[\Delta]{Pt/Ni} \mathop{(CH_2)_6}\limits_{\text{环己烷}} Ph+3H2​Pt/NiΔ​环己烷(CH2​)6​​

Cu/Ag—醇的催化氧化 ​

2CH2CH2OH+O2→ΔCu/Ag2CH3CHO+2H2O\rm 2CH_2CH_2OH + O_2 \xrightarrow[\Delta]{Cu/Ag} 2CH_3CHO + 2H_2O 2CH2​CH2​OH+O2​Cu/AgΔ​2CH3​CHO+2H2​O

醛/酮的还原 ​

CH3CHO+H2→Δ催化剂CH2CH2OH\rm CH_3CHO + H_2 \xrightarrow[\Delta]{\text{催化剂}} CH_2CH_2OH CH3​CHO+H2​催化剂Δ​CH2​CH2​OH

醛的催化氧化 ​

2CH3CHO+O2→Δ催化剂2CH2CHOOH\rm 2CH_3CHO + O_2 \xrightarrow[\Delta]{\text{催化剂}} 2CH_2CHOOH 2CH3​CHO+O2​催化剂Δ​2CH2​CHOOH

Ag—环氧乙烷 ​

2CH2=CH2+O2→ΔAg2C2H4O环氧乙烷\rm 2CH_2{=}CH_2 + O_2 \xrightarrow[\Delta]{Ag} 2\mathop{C_2H_4O}\limits_{\text{环氧乙烷}} 2CH2​=CH2​+O2​AgΔ​2环氧乙烷C2​H4​O​

酚醛树脂 ​

酚醛树脂的制备分为酸性条件下的和碱性条件下的，但高中阶段只含有酸性条件的反应方程式。

该反应可以分为两步。

Ph−OH+HCHO→ΔH+Ph(−OH)−CH2OH邻羟基苯甲醇(1)\tag{1}\rm Ph{-}OH + HCHO \xrightarrow[\Delta]{H^{+}} \mathop{Ph{(-OH)}{-CH_2OH}}\limits_{\text{邻羟基苯甲醇}} Ph−OH+HCHOH+Δ​邻羟基苯甲醇Ph(−OH)−CH2​OH​(1)

nPh(−OH)−CH2OH邻羟基苯甲醇→ΔH+H [−Ph(−OH)−CH2−]nOH+(n−1) H2O(2)\tag{2}\rm n\mathop{Ph{(-OH)}{-CH_2OH}}\limits_{\text{邻羟基苯甲醇}} \xrightarrow[\Delta]{H^{+}} H\space{[\mathllap{-}Ph{(-OH)}{-}CH_2\mathrlap{-}]}_nOH + (n-1)\space H_2O n邻羟基苯甲醇Ph(−OH)−CH2​OH​H+Δ​H [−Ph(−OH)−CH2​−]n​OH+(n−1) H2​O(2)

在碱性条件下，第 (1)(1)(1) 步可能会生成多元羟甲基苯酚，第 (2)(2)(2) 步生成的往往是网状结构的酚醛树脂。

加热+加压+催化剂 ​

工业反应一般条件都较为复杂

工业制乙醇 ​

CH2=CH2+H2O→Δ, 加压催化剂CH3CH2OH\rm CH_2{=}CH_2 + H_2O \xrightarrow[\Delta,\space\text{加压}]{\text{催化剂}} CH_3CH_2OH CH2​=CH2​+H2​O催化剂Δ, 加压​CH3​CH2​OH

工业制乙二醇 ​

C2H4O环氧乙烷+H2O→Δ, 加压催化剂CH2OHCH2OH乙二醇\rm \mathop{C_2H_4O}\limits_{\text{环氧乙烷}} + H_2O \xrightarrow[\Delta,\space\text{加压}]{\text{催化剂}} \mathop{CH_2OHCH_2OH}\limits_{\text{乙二醇}} 环氧乙烷C2​H4​O​+H2​O催化剂Δ, 加压​乙二醇CH2​OHCH2​OH​

光照 ​

光照条件下发生的反应仅有一种——烃基上的卤代

CH4+Cl2→光照CH3Cl+HCl\rm CH_4 + Cl_2 \xrightarrow{\text{光照}} CH_3Cl + HCl CH4​+Cl2​光照​CH3​Cl+HCl

仅催化剂 ​

苯环卤代 ​

Ph+Br2→FeBr3Ph−Br+HBr↑\rm Ph + Br_2 \xrightarrow{FeBr_3} Ph{-}Br + HBr \uparrow Ph+Br2​FeBr3​​Ph−Br+HBr↑

此处 Br2\rm Br_2Br2​ 为纯液溴，Fe\rm FeFe 与 Br2\rm Br_2Br2​ 反应时会放出大量热，故无需加热便可进行。

石油裂化 ​

C8H18→催化剂C4H10+C4H8\rm C_8H_{18} \xrightarrow{\text{催化剂}} C_4H_{10} + C_4H_8 C8​H18​催化剂​C4​H10​+C4​H8​

nCH2=CH2→催化剂 [−CH2−CH2−]n\rm nCH_2{=}CH_2 \xrightarrow{\text{催化剂}}\space {[\mathllap{-}CH_2{-}CH_2\mathrlap{-}]}_n nCH2​=CH2​催化剂​ [−CH2​−CH2​−]n​

CH3CHO+HCN→催化剂CH3CH(OH)CN\rm CH_3CHO + HCN \xrightarrow{\text{催化剂}} CH_3CH(OH)CN CH3​CHO+HCN催化剂​CH3​CH(OH)CN

2CH3CHO→催化剂CH3CβH(OH)CαH2CHO\rm 2CH_3CHO \xrightarrow{\text{催化剂}} CH_3\overset{\beta}{C}H(OH)\overset{\alpha}{C}H_2CHO 2CH3​CHO催化剂​CH3​CβH(OH)CαH2​CHO

CH4→高温C+2H2\rm CH_4 \xrightarrow{\text{高温}} C + 2H_2 CH4​高温​C+2H2​

无条件 ​

CH2=CH2+Br2⟶CH2BrCH2Br\rm CH_2{=}CH_2 + Br_2 \longrightarrow CH_2BrCH_2Br CH2​=CH2​+Br2​⟶CH2​BrCH2​Br

CH2=CHCH=CH2+Br2⟶CH2BrCHBrCH=CH2\rm CH_2{=}CHCH{=}CH_2 + Br_2 \longrightarrow CH_2BrCHBrCH{=}CH_2 CH2​=CHCH=CH2​+Br2​⟶CH2​BrCHBrCH=CH2​

CH2=CHCH=CH2+Br2⟶CH2BrCH=CHCH2Br\rm CH_2{=}CHCH{=}CH_2 + Br_2 \longrightarrow CH_2BrCH{=}CHCH_2Br CH2​=CHCH=CH2​+Br2​⟶CH2​BrCH=CHCH2​Br

CH≡CH+2Br2⟶CHBr2CHBr2\rm CH{\equiv}CH + 2Br_2 \longrightarrow CHBr_2CHBr_2 CH≡CH+2Br2​⟶CHBr2​CHBr2​

醇/酚和钠 ​

2CH3CH2OH+2Na⟶2CH3CH2ONa+H2↑\rm 2CH_3CH_2OH + 2Na \longrightarrow 2CH_3CH_2ONa + H_2\uparrow 2CH3​CH2​OH+2Na⟶2CH3​CH2​ONa+H2​↑

2Ph−OH+2Na⟶2Ph−ONa+H2↑\rm 2Ph{-}OH + 2Na \longrightarrow 2Ph{-}ONa + H_2\uparrow 2Ph−OH+2Na⟶2Ph−ONa+H2​↑

CH3COOH+NaOH⟶CH3COONa+H2O\rm CH_3COOH + NaOH \longrightarrow CH_3COONa + H_2O CH3​COOH+NaOH⟶CH3​COONa+H2​O

Ph−OH+NaOH⟶Ph−ONa+H2O\rm Ph{-}OH + NaOH \longrightarrow Ph{-}ONa + H_2O Ph−OH+NaOH⟶Ph−ONa+H2​O

强氧化剂 ​

烯炔、苯环上的烃基（且与苯环相连的碳上含氢）、醇、酚、醛、草酸均可使 KMnO4/H+\rm KMnO_4/H^{+}KMnO4​/H+ 褪色，其中烯烃和苯的氧化经常被用作所短碳链。

其中醇还可使 K2Cr2O7\rm K_2Cr_2O_7K2​Cr2​O7​ 从● 橙色 变为● 灰绿色

显色 ​

仅有苯酚遇 FeCl3\rm FeCl_3FeCl3​ 变● 紫色，在空气中变● 粉红色
